# ecommerce-retail-analytics
End-to-end retail analytics project using Python, SQL, Power BI

## Project Overview
End-to-end retail analytics project analyzing 110,000+ real e-commerce 
transactions to uncover sales trends, customer behavior, and business insights 
using Python, SQL, and Power BI.

## Tools & Technologies
- **Python** (pandas, matplotlib) — Data cleaning & EDA
- **SQL** (SQLite) — Business queries & analysis
- **Power BI** — Interactive dashboard
- **Google Colab** — Development environment

## Project Structure
- `notebooks/01_data_cleaning.ipynb` — Data cleaning & merging
- `notebooks/02_eda.ipynb` — Exploratory data analysis & charts
- `notebooks/03_sql_analysis.ipynb` — SQL business queries
- `notebooks/04_rfm_segmentation.ipynb` — Customer segmentation
- `sql/queries.sql` — All SQL queries
- `ecommerce_dashboard.pdf` — Power BI dashboard export

## Key Insights
- Revenue grew 3x from 2016 to 2018
- Top revenue categories: Health & Beauty, Watches, Bed & Bath
- São Paulo state accounts for highest number of orders
- RFM segmentation identified Champions, Loyal, At-Risk & Lost customers
- Average delivery time varies significantly across states

## Dashboard
3-page interactive Power BI dashboard covering:
- Sales Overview (Revenue, Orders, Avg Order Value)
- Product Analysis (Top categories, State-wise orders)
- Customer Segments (RFM analysis breakdown)

## Dataset
Brazilian E-Commerce Public Dataset by Olist — available on Kaggle
100,000+ orders from 2016-2018
